# Phase 8: Cleanup and testing

## Goal

Final cleanup, testing, and ensure everything works together.

## Tasks

1. Remove any temporary code, TODOs, debug prints
2. Fix all warnings
3. Ensure all tests pass
4. Test debug UI displays correctly:
   - Textures show images and metadata
   - Shaders show code and status
   - Fixtures show details and previews
5. Format code with `cargo +nightly fmt`
6. Move plan to `_done`

## Success Criteria

- No temporary code or TODOs
- All warnings fixed
- All tests pass
- Debug UI displays all node information correctly
- Code is clean and readable
- Code is formatted
- Plan moved to `_done`
