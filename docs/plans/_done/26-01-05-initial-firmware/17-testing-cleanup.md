# Phase 17: Testing and cleanup

## Goal

Final testing, cleanup, and verification.

## Tasks

1. Remove any temporary code, TODOs, debug prints
2. Fix all warnings
3. Ensure all tests pass (if any exist)
4. Ensure all code is clean and readable
5. Verify acceptance criteria:
   - Can send/receive JSON commands
   - Can load/store project
   - Can log messages
   - Framework is established for future shader execution
6. Move plan directory to `lightplayer/plans/_done/`
7. Run `cargo +nightly fmt` to format code in `lightplayer/` directory

## Success Criteria

- No temporary code or TODOs remain
- All warnings fixed
- All tests pass
- Code is clean and readable
- Acceptance criteria verified
- Plan moved to `_done/`
- Code formatted
